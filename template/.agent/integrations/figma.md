# Figma Integration

Use this file when the project uses Figma for UI design, design systems or UX flows.

## Allowed read operations

```text
- inspect frames
- read component names
- read design tokens
- review user flows
- compare UI implementation with design intent
```

## Write operations requiring approval

```text
- creating comments
- updating design documentation
```

## Forbidden by default

```text
- modifying design files
- deleting components
- exposing private design links
```

## Agent usage

Use Figma context for:

```text
- UX review
- UI consistency
- accessibility checks
- component mapping
- implementation guidance
```
