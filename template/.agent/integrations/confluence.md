# Confluence Integration

Use this file when the project uses Confluence for documentation, decisions, meeting notes or requirements.

## Allowed read operations

```text
- read pages
- read page trees
- read decision records
- read meeting notes
```

## Write operations requiring approval

```text
- create pages
- update pages
- add comments
- move pages
```

## Forbidden by default

```text
- deleting pages
- changing permissions
- bulk-moving page trees
- publishing sensitive information
```

## Documentation rule

When Confluence is used as a source, summarize and cite the relevant page or export location in generated harness artifacts.
